Steps are similar to statements in programming language.  An exhaustive list of steps can be found in the [steps reference](0.6_steps_api.html).

Example: 

```yaml
- thinkTime:
    duration: 3s
```

```yaml
- set: isOneWay <- false
```

```yaml
- httpRequest: ...
```
