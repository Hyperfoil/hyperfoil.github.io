Handler manages processing of HTTP responses.  It can process returned body, header, etc.  For more information, please refere to the [handler references](0.6_handler.html).

```yaml
- httpRequest:
    handler:
      HANDLER_PROPERTIES
```

Example:

```yaml
- httpRequest:
    POST:
      value: /flight/queryflights
    handler:
      body:
        json:
          query: .[] 
```
