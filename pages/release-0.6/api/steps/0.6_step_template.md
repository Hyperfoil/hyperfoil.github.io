Format pattern into session variable. 

| Property | Type | Description |
| ------- | ------- | -------- |
| pattern | String | Pattern to be encoded, e.g. <code>foo${variable}bar${another-variable}</code>  |
| toVar | String | Variable name to store the result.  |
